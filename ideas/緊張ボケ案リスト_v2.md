# 緊張ネタ - ボケ・ツッコミ案リスト v3

## 設計思想

**核心**: 実在する緊張対策を「ズラす・変換する」ことで笑いを生む

```
例: 「人」を3回書いて飲み込む（実在）
  → 「犬」を3回書いて飲み込む（ズラし）
  → 犬×3 = ケルベロス（変換の妙）
```
