### 見た目・声

### ボケの傾向

- **得意なボケ**:

- **ボケのパターン**:

### 基本設定

- **名前**: [未定]

### しゃべり方の特徴

- **口調**: 批判的、ストレート、ワードで笑かす
- **テンポ**: やや早口
- **トーン**: 熱くなりやすい、でも冷静な批判も

### ツッコミの傾向

- **得意なツッコミ**:
  - **大喜利ツッコミ**: ワードで笑かす、例えが面白い
  - **気持ちいいツッコミ**: 「〇〇やんけ！」がビシッと決まる
  - **批判ボケツッコミ**: ツッコミ自体がダークでボケになってる

- **ツッコミのパターン**:
  - パターン1: 「〇〇やんけ！」系（気持ちよく断定）
  - パターン2: ダークな批判（「全員詐欺師やろ」「誰が得するねん」）
  - パターン3: 例えツッコミ（「あんま喋らんコンビニ店員はつんくか？」）

### 批判ボケの例（普段の会話から）

- 「jpopがダサい」
- 「ジャニーズタレントはちんちん触らせてテレビ出てていいのか」
- 「あんま喋らんコンビニ店員はつんくか？」

## コンビの関係性

### 立ち位置

- **力関係**: 対等
- **距離感**: 同級生、仲良し
- **関係性**: 互いを理解している、遠慮なく言い合える

### 掛け合いの特徴

- **テンポ感**: ボケ（ゆっくり）→ ツッコミ（早口）のコントラスト
- **ボケとツッコミの比率**: 柔軟（場面による）
- **得意な展開パターン**:
  1. ボケが提案 → ツッコミが切る → 往復
  2. 会話の中で自然にズレていく
  3. 一個のボケを掘り下げる
  4. 次々新しいボケを出す

### ビジュアルの対比

| ボケ | ツッコミ |
|------|---------|
| すらっと好青年 | スキンヘッドでいかつい |
| 低音ゆっくり | 普通の声で早口 |
| 冷静に極端なこと言う | 熱く批判する |

→ ギャップが武器になる

### このコンビならではの強み

1. **ボケの理屈暴走**: 冷静に狂気的な結論まで持っていく
2. **ツッコミの批判ボケ**: ツッコミ自体が笑える、ダークな角度
3. **ビジュアルギャップ**: 見た目と中身の裏切り
4. **ダークさOK**: 攻めた題材も成立する

### 意識すべきポイント

- ボケが理屈を展開する「間」を大事に
- ツッコミは気持ちよく切るタイミングを見極める
- 批判ボケは攻めすぎると引かれるラインを意識
- 二人とも関西弁のトーンを揃える
