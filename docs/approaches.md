# 切り口パターン集

テーマをどう料理するか。このコンビの武器を活かす切り口を整理。

---

## 切り口一覧

### 1. 正論の極端化

**概要**: 社会的に正しいこと、正論を極端に推し進めると狂気になる

**なぜこのコンビに合うか**:
- ボケの理屈暴走が最大限活きる
- 最初は「確かに」と思わせてから裏切る

**展開パターン**:
```
正論を提示 → 論理的に展開 → どんどん極端に → 狂気的結論
```

**具体例**:
| 正論 | 極端化 | 狂気的結論 |
|------|--------|-----------|
| 差別をなくそう | 全員平等に扱う | 全員平等に嫌う |
| 環境に配慮しよう | CO2削減 | 人間が一番環境に悪い、人間削減 |
| 多様性を認めよう | あらゆる価値観を認める | 犯罪者の価値観も認める |
| 効率化しよう | 無駄を省く | 人間関係も効率化、感情も無駄 |

**使えるテーマ**:
- 「正しい人間になりたい」
- 「SDGsを本気で考える」
- 「差別をなくす方法」

---

### 2. システムへの疑問

**概要**: 当たり前に存在するシステムや制度を根本から疑う

**なぜこのコンビに合うか**:
- ツッコミの批判精神が活きる
- 「そもそも」から始められる

**展開パターン**:
```
システムを提示 → 「そもそもこれ何？」 → 定義が破綻 → 「いらんくない？」
```

**具体例**:
| システム | 疑問 | 破綻ポイント |
|----------|------|-------------|
| 学校 | なぜ全員同じことを学ぶ？ | 使わない知識が多すぎる |
| 会社 | なぜ毎日同じ場所に集まる？ | 効率悪い |
| 結婚 | なぜ国に届け出る？ | 愛に書類いらん |
| 選挙 | なぜ多数決？ | 多数が正しいとは限らん |
| 敬語 | なぜ年上に敬意？ | 年上がえらいとは限らん |

**使えるテーマ**:
- 「日本から敬語をなくす提案」
- 「会社という制度」
- 「本音と建前」

---

### 3. 人間観察の異常視点

**概要**: 「こういう奴いるよな」を異常な角度から分析する

**なぜこのコンビに合うか**:
- ツッコミの「許せない奴」への批判
- ボケの分析力
- ダークな人間観が出せる

**展開パターン**:
```
あるあるを提示 → 異常な分析 → 「つまりこういうことやろ」 → ダークな結論
```

**具体例**:
| あるある | 異常な分析 |
|----------|-----------|
| 優しい人 | 優しさ = 自己満足、見返り期待 |
| コミュ力高い人 | コミュ力 = 操作能力、詐欺師と同じ |
| 前向きな人 | 前向き = 現実逃避 |
| 空気読む人 | 空気読む = 自分がない |

**使えるテーマ**:
- 「優しさの正体」
- 「コミュ力とは何か」
- 「許せない奴リスト」

---

### 4. 概念の再定義

**概要**: 曖昧な概念を理屈で定義しようとして破綻する

**なぜこのコンビに合うか**:
- ボケの理屈こねが発揮される
- 定義の試行錯誤で笑いが取れる

**展開パターン**:
```
概念を提示 → 定義を試みる → 全部破綻 → 「存在しないのでは」
```

**具体例**:
| 概念 | 定義の試み | 破綻 |
|------|-----------|------|
| 大人 | 経済的自立 | ニートは子供？ |
| 大人 | 責任を取る | 政治家は子供？ |
| 幸せ | 欲望が満たされる | 欲望なければ幸せ？ |
| 友達 | 損得なしの関係 | 完全に損得なしは無理 |

**使えるテーマ**:
- 「大人になるとは」
- 「幸せの定義」
- 「友達とは何か」

---

### 5. 効率化・最適化

**概要**: 本来効率化すべきでないものを効率化しようとする

**なぜこのコンビに合うか**:
- ボケの理屈暴走の王道パターン
- ビジネス用語でギャップを作れる

**展開パターン**:
```
効率化対象を提示 → 数値化・分析 → 最適化提案 → 人間味がなくなる
```

**具体例**:
| 対象 | 効率化 | 問題 |
|------|--------|------|
| 友達作り | コスパ分析、損切りライン | 人間関係が株取引 |
| 恋愛 | マッチング最適化、ROI | 愛がない |
| 葬式 | 時間短縮、コスト削減 | 弔いがない |
| 謝罪 | マニュアル化、角度設定 | 心がない |

**使えるテーマ**:
- 「人間関係コスパ理論」
- 「効率的な友達の作り方」
- 「謝罪会見の正解」

---

### 6. 逆張りの逆張り

**概要**: 世間の逆張りをする → その逆張りの逆張り → 結局どこにも着地しない

**なぜこのコンビに合うか**:
- ツッコミの批判精神
- 二人でどんどん批判し合える

**展開パターン**:
```
一般論 → 逆張り（ツッコミ）→ 逆張りの逆張り（ボケ）→ 混乱
```

**具体例**:
```
一般: 「努力は大事」
逆張り: 「努力しても報われんやつおるやろ」
逆逆張り: 「でも努力せんと何も始まらん」
さらに: 「でも努力の方向性間違ってたら無駄」
結局: 「何が正解やねん」
```

---

### 7. 時代錯誤の更新

**概要**: 古い価値観を現代にアップデートしようとするが、方向がおかしい

**なぜこのコンビに合うか**:
- 社会風刺ができる
- 「アップデート」という概念で笑える

**展開パターン**:
```
古い価値観 → 「アップデートしよう」 → おかしな方向に更新 → 余計おかしくなる
```

**具体例**:
| 古い価値観 | アップデート | おかしな方向 |
|-----------|-------------|-------------|
| 男らしさ | 現代の男らしさ | 稼がない、家事する、でも強くない |
| 年功序列 | 能力主義 | でも能力の定義が曖昧 |
| 根性論 | 科学的トレーニング | でも科学も間違う |

## 切り口の組み合わせ

| 組み合わせ | 効果 |
|-----------|------|
| 正論極端化 + 効率化 | 「正しいことを効率的にやる」→ 狂気 |
| システム疑問 + 概念再定義 | 「そもそも○○って何」から制度批判 |
| 人間観察 + 効率化 | 人間を分析・数値化 |

## M-1向け切り口の選び方

### 1回戦で使いやすい切り口

| 切り口 | 理由 |
|--------|------|
| 効率化・最適化 | 分かりやすい、ビジネス用語でインパクト |
| システムへの疑問 | 誰でも「確かに」と思える |
| 人間観察 | あるあるベースで入りやすい |

### 避けた方がいい切り口（1回戦では）

| 切り口 | 理由 |
|--------|------|
| 概念再定義 | 抽象的すぎる可能性 |
| 逆張りの逆張り | 複雑、着地しにくい |
