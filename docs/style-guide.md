# しゃべくり漫才スタイルガイド

## しゃべくり漫才とは

しゃべくり漫才は、掛け合いのテンポとリズムを重視した漫才スタイルです。

### 核となる要素

1. **スピード感**: 素早い掛け合いによる畳み掛け
2. **リズム**: ボケとツッコミの心地よい反復
3. **密度**: 短時間に多くの笑いを詰め込む
4. **テンション**: 勢いを維持したまま最後まで駆け抜ける

---

## テンポ管理の原則

### 基本リズム

```
ボケ（2-3秒） → ツッコミ（1-2秒） → ボケ（2-3秒） → ツッコミ（1-2秒）
```

- **1つのボケは短く**: 長い説明は避け、核心だけ伝える
- **ツッコミは即座に**: ボケの直後、0.5秒以内に入る
- **畳み掛けの連続**: 3-5回の掛け合いをワンセットとする

### テンポの緩急

| セクション | テンポ | 目的 |
|----------|--------|------|
| 導入部 | 普通〜やや早 | 観客を引き込む |
| 展開部 | 早い | 笑いの密度を上げる |
| クライマックス | 最速 | 畳み掛けて爆発させる |
| オチ | 一瞬の間 → 締め | 余韻を残す |

---

## 構成の黄金律

### 時間配分（3分ネタの場合）

```
0:00-0:30  導入（つかみ）      - テーマ提示 + 軽いボケ
0:30-1:30  展開1（ボケの連打）  - 本題に入り、リズムを作る
1:30-2:30  展開2（加速）        - テンポアップ、畳み掛け
2:30-2:50  クライマックス       - 最大の笑いどころ
2:50-3:00  オチ                - 綺麗に締める
```

### ネタの密度

- **1分あたり**: ボケ 8-12個
- **ツッコミ**: ボケと同数以上
- **笑いどころ**: 10-15秒に1回（観客の笑い声想定）

---

## ボケのパターン

### しゃべくり漫才に適したボケ

✅ **推奨**:
- **誇張ボケ**: 「めちゃくちゃ〜」「超〜」などの極端な表現
- **勘違いボケ**: 話の流れを誤解して斜め上の返答
- **言い換えボケ**: 予想外の言葉で言い直す
- **連想ボケ**: トピックから連想ゲーム的に飛躍
- **数字ボケ**: 具体的すぎる/大げさな数値

❌ **非推奨**:
- 長い説明が必要なボケ（テンポが止まる）
- 複雑な設定や前提知識が必要なボケ
- 一発ネタ系（繰り返せない）

### ボケの連打テクニック

```
例：「朝ごはん」をテーマにした場合

ボケ: 今日の朝ごはん、めちゃくちゃ豪華だったんですよ
ツッコミ: おお、何食べたの？
ボケ: フルコース！
ツッコミ: 朝から？
ボケ: 前菜からデザートまで27品！
ツッコミ: 多すぎるわ！何時間かかんねん
ボケ: 3時間半
ツッコミ: それ昼飯やろ！
```

**ポイント**:
- ボケが加速度的に大きくなる（豪華 → フルコース → 27品 → 3時間半）
- ツッコミは短く的確に
- リズムが途切れない

---

## ツッコミのパターン

### しゃべくり向けツッコミ

| タイプ | 特徴 | 使用例 |
|-------|------|--------|
| **否定ツッコミ** | ボケを即座に否定 | 「違うわ！」「ならんわ！」 |
| **疑問ツッコミ** | 論理的におかしい点を指摘 | 「なんでやねん」「どういうこと？」 |
| **確認ツッコミ** | わざと確認して畳み掛ける | 「いや、ちょっと待て」「え、マジで？」 |
| **要約ツッコミ** | ボケを端的にまとめる | 「つまり〜ってことやろ？」 |
| **切り替えツッコミ** | 話を先に進める | 「それはええとして」「で？」 |

### ツッコミの強弱

```
弱 →→→→→ 強

「え？」 < 「いやいや」 < 「ちょっと待て」 < 「何言ってんの！」 < 「ありえへんやろ！」
```

**使い分け**:
- 序盤: 弱〜中のツッコミでリズムを作る
- 中盤: 中〜強で盛り上げる
- クライマックス: 最強のツッコミで締める

## 間の取り方

### 基本原則

しゃべくり漫才では**極力間を作らない**のが原則ですが、効果的な間もあります。

✅ **効果的な間**:
- **オチの前**: 0.5-1秒の溜め（期待感を高める）
- **大きなボケの後**: ツッコミ待ちの間（0.3秒程度）
- **展開の切り替え**: 話題転換時の一呼吸（0.5秒）

❌ **避けるべき間**:
- 言葉に詰まる（準備不足に見える）
- 次の台詞を考える時間（リズムが崩れる）
- ボケとツッコミの間が空きすぎる（1秒以上）

## 言葉選びのコツ

### テンポを加速する言葉

- **短い単語**: 「超」「めっちゃ」「マジ」
- **リズミカルな表現**: 「バババッと」「ガーッと」「ドドドッと」
- **数字**: 具体的な数値で勢いをつける

### テンポを殺す言葉（避ける）

- 「〜みたいな感じで」（曖昧で冗長）
- 「えーっと」「なんていうか」（迷いが見える）
- 長い接続詞や前置き

## ネタ構成のチェックリスト

制作時に確認すべきポイント：

### テンポ
- [ ] 1分あたりボケ8-12個あるか
- [ ] ボケとツッコミの間が適切か（1秒以内）
- [ ] テンポの緩急がついているか

### 内容
- [ ] 導入でテーマが明確か
- [ ] ボケが連打されているか
- [ ] ツッコミが的確か
- [ ] クライマックスで最大の笑いがあるか
- [ ] オチが明確か

### キャラクター
- [ ] ボケ担当のキャラに合っているか
- [ ] ツッコミ担当のキャラに合っているか
- [ ] 二人の関係性が表現できているか

### 技術
- [ ] 専門用語や難しい言葉を避けているか
- [ ] 万人にわかる内容か
- [ ] 時事ネタの賞味期限は大丈夫か

## NGパターン

### 構成上のNG

❌ **説明漫才**: 状況説明が長すぎる
❌ **一人語り**: ボケが長く話しすぎてツッコミが入れない
❌ **ツッコミ不在**: ボケっぱなしで収拾がつかない
❌ **中だるみ**: 中盤でテンポが落ちる
❌ **オチ無し**: なんとなく終わる

### 内容上のNG

❌ **内輪ネタ**: 限られた人しかわからない
❌ **知識前提**: 特定の知識が必要
❌ **説明が必要**: 「〜って知ってます？」から始まるネタ
❌ **暗いネタ**: 病気、死、差別など
❌ **攻撃的**: 特定の誰かを傷つける内容

## 参考スタイル

しゃべくり漫才の参考となる要素：

### リズムの参考
- テンポの良い掛け合い
- 畳み掛けるようなボケの連打
- 間髪入れずのツッコミ

### 構成の参考
- 明確なテーマ設定
- 加速していく展開
- クライマックスからの綺麗な締め

## 練習方法

### テンポ感の訓練

1. **タイマー練習**: 3分きっかりで終わる練習
2. **速度変更**: あえて倍速で読んでみる
3. **録音チェック**: 間が空いている箇所を発見
4. **メトロノーム**: 一定リズムを保つ訓練

### ネタ磨き

1. **削る**: 無駄な言葉を徹底的に削除
2. **入れ替える**: より短い言葉に置き換え
3. **並べ替える**: ボケの順番を最適化
4. **加える**: 足りない箇所にボケを追加
