# 漫才構造パターン集

このコンビの武器を活かす構造パターンを整理。ネタ作りの際に参照する。

---

## 基本構造

### A. 逆転構造

**概要**: 通常の「ボケ→ツッコミ」の流れを逆転させる

**流れ**:
```
ツッコミが先に批判を展開
    ↓
ボケが「いや、それは○○という観点では合理的」と理屈擁護
    ↓
ツッコミ「何擁護してんねん！」
    ↓
ボケの擁護がどんどん極端に
    ↓
ツッコミ「お前が一番おかしいわ！」
```

**活かせる武器**:
- ツッコミの批判ボケ（最初から批判できる）
- ボケの理屈擁護（意外性がある）

**向いてるテーマ**:
- 「許せない奴」系
- 社会批判系
- 人間観察系

**例**:
```
ツッコミ: 俺、許せない奴おるねん
ボケ: 誰？
ツッコミ: 電車で足広げて座る奴
ボケ: いや、それは生物学的に合理的やで。オスは縄張り意識が...
ツッコミ: 何擁護してんねん！
```

---

### B. 理屈暴走構造

**概要**: ボケが理屈っぽく何かを説明し、どんどん極端になっていく

**流れ**:
```
ボケが「○○について考えた」と提案
    ↓
理屈っぽく説明開始
    ↓
どんどん極端な方向へ
    ↓
ツッコミが「待て待て」と止める
    ↓
ボケが論破する or さらに暴走
    ↓
狂気的な結論に到達
    ↓
ツッコミ「お前がおかしいわ！」
```

**活かせる武器**:
- ボケの理屈こね能力
- 冷静に狂気を語るギャップ

**向いてるテーマ**:
- 「○○を最適化する」系
- 「○○の正解」系
- 定義・分析系

**例**:
```
ボケ: 友達を効率的に作る方法、考えたんやけど
ツッコミ: お、ええやん
ボケ: まず、友達候補を数値化する
ツッコミ: 数値化？
ボケ: コスパで考えるねん。時間対効果で
ツッコミ: 友達を株みたいに言うな
ボケ: 損切りラインも設定して...
ツッコミ: 損切り！？
```

---

### C. 批評構造

**概要**: 二人で何かを批評する形式で、批評自体がボケになる

**流れ**:
```
題材を提示（映画、音楽、社会現象など）
    ↓
ボケが斜め上の批評
    ↓
ツッコミが「そこちゃうやろ」
    ↓
ツッコミも批評に参加 → ダークな方向へ
    ↓
二人の批評がエスカレート
    ↓
元の題材から逸脱
```

**活かせる武器**:
- ツッコミの批判精神
- ボケの独特な視点
- 二人とも「批判できる」強み

**向いてるテーマ**:
- 文化批評（音楽、映画、SNS）
- 社会現象批評
- 業界批評

**例**:
```
ボケ: 最近の映画、どう思う？
ツッコミ: まあ、色々あるけど
ボケ: 俺、ヒーロー映画が理解できへん
ツッコミ: なんで？
ボケ: 悪役を倒して終わるやん。根本的な解決になってない
ツッコミ: 映画に社会問題の解決求めんな
ボケ: でも悪役が生まれる社会構造を...
ツッコミ: それはそうやな...社会が悪いわ、全部
```

---

### D. シミュレーション構造

**概要**: 「もしも〇〇だったら」を二人でシミュレーションする

**流れ**:
```
ボケが仮説を提示
    ↓
シミュレーション開始
    ↓
どんどんおかしな方向に
    ↓
ツッコミが現実に引き戻そうとする
    ↓
ボケが「でも○○」とさらに展開
    ↓
シミュレーションが崩壊
```

**活かせる武器**:
- ボケの論理展開力
- 二人の掛け合いで世界を作る

**向いてるテーマ**:
- 「もしも○○がなかったら」
- 「全員○○だったら」
- 「○○が〇〇したら」

**例**:
```
ボケ: 本音と建前、もうやめへん？
ツッコミ: 急にどうした
ボケ: 全員本音で生きる社会をシミュレーションしてみ
ツッコミ: ほな、会社で「この仕事意味ない」って言えるな
ボケ: そうそう。「お前嫌い」も言える
ツッコミ: 社会崩壊するやんけ
ボケ: 崩壊した方がええんちゃう
ツッコミ: お前の本音、重すぎるわ
```

---

### E. 概念解体構造

**概要**: 当たり前の概念を理屈で解体していく

**流れ**:
```
日常的な概念を提示
    ↓
ボケが「それって何？」と疑問
    ↓
定義を試みる → 全部破綻
    ↓
ツッコミ「じゃあ○○って何やねん」
    ↓
ボケ「存在しないんちゃう」
    ↓
ニヒリズムに到達
```

**活かせる武器**:
- ボケの分析力
- ツッコミの「確かに」という共感

**向いてるテーマ**:
- 「幸せとは」「大人とは」「友達とは」
- 抽象概念全般

---

## 構造の組み合わせ

実際のネタでは複数の構造を組み合わせることが多い：

| 組み合わせ | 効果 |
|-----------|------|
| 逆転 + 理屈暴走 | ツッコミの批判にボケが理屈で暴走擁護 |
| 批評 + シミュレーション | 批評から「じゃあこうしたら」に展開 |
| 概念解体 + 理屈暴走 | 定義を探るうちに極端な結論へ |

## M-1向け構造の注意点

### 1回戦で重要なこと

1. **最初の30秒で掴む**
   - 設定説明は短く
   - すぐに笑いを入れる
   - 二人のキャラを見せる

2. **構造が分かりやすい**
   - 複雑な設定は避ける
   - 「何をやってるか」が伝わる

3. **4分で起承転結**
   - 導入（30秒）
   - 展開1（1分）
   - 展開2・加速（1分30秒）
   - クライマックス・オチ（1分）

4. **このコンビのニンを見せる**
   - ボケの理屈暴走を見せる
   - ツッコミの批判ボケを見せる
   - 二人の関係性が伝わる
