## ネタの設定

### コンセプト
ネタを披露するのに緊張しているという設定で、緊張をほぐす方法を次々と提案するが、その方法がどんどんおかしくなっていく。誇張ボケと勘違いボケを中心に、テンポよく畳み掛ける。

### 前提条件
- 特になし（誰でもわかる緊張の話）
- 「犬を書いて飲み込む」という緊張対策は一般的に知られている

### ネタの流れ（簡易版）
1. 導入: ボケが緊張していると告白、ツッコミが「それが仕事やろ」
2. 展開1: 定番の緊張対策を提案するが、微妙にズレている（犬×3=ケルベロスなど）
3. 展開2: さらにエスカレート、人を何かに見立てる系ボケ（ジャガイモ→ボーリング玉→ミスターノー）
4. クライマックス: もう何がなんだかわからないレベルの緊張対策
5. オチ: 「もう始まってるやん」で締める

## 演出メモ

### 重要なポイント
- [ ] 「ケルベロスか！」: 強めに言って笑いを取る
- [ ] 「ミスターノーか！」: 元ネタがわかる人に刺さるように
- [ ] 緊張してるキャラを最初から最後まで維持（ボケが常にソワソワ）

### 身振り・動き
- ボケは緊張してる様子（落ち着きなく）
- ツッコミは冷静に、時々呆れた仕草

### トーン・声の使い方
- ボケ: やや早口で、不安そうな感じ
- ツッコミ: しっかりした関西弁、メリハリつける

## 分析

### ボケの数
- 導入: 1個（緊張すると言う）
- 展開1: 4個（犬×3=ケルベロス、深呼吸100回、世界征服イメージ、ジャガイモ）
- 展開2: 4個（ボーリング玉=ミスターノー、温泉、北向いて回転、宇宙）
- クライマックス: 2個（台本10000回、緊張してる場合じゃない）
- **合計**: 11個

### ツッコミの数
- **合計**: 15個以上（ボケに対して必ずツッコミ）

### テンポ分析
- 1分あたりボケ数: 約3.7個（11個÷3分）
- ボケ:ツッコミ比率: 約1:1.4
- 平均的な掛け合いの間隔: 5-10秒程度

### スタイルガイド準拠チェック
- [ ] 1分あたりボケ8-12個 → **要改善**（現状3.7個、もう少しボケを追加推奨）
- [x] テンポの緩急がついている（導入→展開で加速→クライマックス）
- [x] クライマックスが明確（宇宙、10000回暗記）
- [x] オチが綺麗に締まっている（メタネタとして成立）
- [x] NGパターンを避けている

### 改善提案
- 展開1と展開2にそれぞれ2-3個ずつボケを追加すると、より密度が高まる
- 特に展開1（0:30-1:30）は1分間あるので、6-8個のボケがあると理想的
